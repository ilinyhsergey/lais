1) "mvn clean build"
    - to build runnable jar file ./target/LaIS-1.0-SNAPSHOT.jar
2) "java -jar target/LaIS-1.0-SNAPSHOT.jar data.txt"
    - to run jar file.
    - data.txt - should contain a sequence of numbers separated by whitespaces.
3) To get statistics you can run com.ilinykh.algo.AppTest and view files: ./result-5000.csv and ./result-5000-avg-std.csv
